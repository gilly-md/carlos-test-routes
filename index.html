How to Use

Each route has two buttons. Pick whichever suits your setup.

◎

Google Maps

  • Tap View on Google Maps on your phone to open the route in the Google Maps app.
  • Press Start for turn-by-turn navigation.
  • Using Apple CarPlay or Android Auto? The navigation mirrors to your car's screen automatically.
⚙

GPX File

  • BMW, Mercedes, VW, Audi — Copy the .gpx file to a USB stick, plug it in, and import via the navigation menu.
  • Garmin — Connect via USB, copy the file to Garmin/GPX, then find it under Where To? > Saved > Routes.
  • TomTom — Import via TomTom MyDrive, or copy directly to the device and use the route planner.
  • Tesla — GPX isn't supported natively. Use the Google Maps link and share the destination via the Tesla app.
Technical note — GPX files use the <rte> format with named <rtept> waypoints at key junctions. Your GPS calculates the driving path between each point automatically. Full turn-by-turn detail is in the tables on each route page.
